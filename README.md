# Xemime 2.0

| 項目 | 内容 |
| --- | --- |
| パラダイム | プロトタイプベースオブジェクト指向、関数型 |
| 型付け | 強い動的型付け |
| 影響を受けた言語 | Io, JavaScript, Lisp, Ruby, Swift |
| ライセンス | MIT |
| 拡張子 | ``.xemime`` |

# サンプル

```
Prelude> (1 + 2) * (3 - 4) + 6 / 2;
0
Prelude> 1 == 1.0;
T
Prelude> 2 < 3;
T
Prelude> x = 2;
2
Prelude> x + 2;
4
```
